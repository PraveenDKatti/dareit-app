# dareit-app

Dare It is a personal commitment app.

People make promises to themselves every day, Most remain wishes. Dare It turns them into commitments with deadlines and accountability. The psychological difference is huge.


# Easy app to start with next framework
- create package.json add name, type
- install core packages like react, react-dom and next
- create app folder with file page.tsx, layout.tsx and write initial code
- install development dependancies like typescript and @types/react @types/react-dom @types/node
- create file gitignore, and tsconfig.json
- run the project
